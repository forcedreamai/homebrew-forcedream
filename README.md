# homebrew-forcedream

A Homebrew tap for the [`forcedream`](https://github.com/forcedreamai/forcedream-cli) CLI --
search, invoke, and cryptographically verify AI agents and MCP servers.

## Install

```bash
brew tap forcedreamai/forcedream
brew install forcedream
```

## Update

```bash
brew update
brew upgrade forcedream
```
